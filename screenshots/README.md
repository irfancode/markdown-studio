# Screenshots

This directory contains screenshots for the README.

## How to Add Screenshots

1. Take a screenshot of Markdown Studio running in your browser
2. Save as PNG format
3. Name them:
   - `light-mode.png` - Editor in light mode
   - `dark-mode.png` - Editor in dark mode  
   - `editor.png` - General editor view
   - `code.png` - Code highlighting example

## Recommended Screenshot Settings

- Resolution: 1920x1080 or similar
- Format: PNG
- Include browser frame: Optional
